#  Sales Dashboard (Power BI)

##  Overview
An interactive dashboard built using Power BI to analyze sales performance and business trends.

##  Tools Used
- Power BI (Data Modeling, Visualization)
- Excel/CSV Dataset

##  KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin

##  Dashboard Features
- Region-wise Sales Analysis
- Category-wise Profit Analysis
- Monthly Sales Trend
- Top Customers

##  Key Insights
- West region is the top-performing region
- Technology category yields highest profit
- Sales show steady growth over time
- Revenue is concentrated among top customers

- dashboard.pbix
